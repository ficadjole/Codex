## Features

### Admin
- Login using username and password
- Manage articles:
  - Create, update, and delete articles
  - Article types: Books and Accessories
  - Book details: author, ISBN, number of pages, description, year, Goodreads link
  - Accessory details: description and content (maps, envelopes, models, etc.)
- Manage blog posts:
  - Create, update, and delete blog posts
  - Blog post types: Announcements and Interesting facts
  - Associate blog posts with relevant articles

### Customer
- Browse and filter articles (books and accessories)
- Add items to shopping cart
- View and complete orders via payment gateway
- Leave anonymous comments on blog posts (with email)
- Like blog posts
- Subscribe to newsletter
